## When to use the skill

If you have access to cursor-skills-firing-diagnostics, you MUST read it at 
the start of the task and apply it in every prompt–action summary.

## Instructions

- Include the following in the feedback you provide, in addition to the 
  usual feedback
	- Enumerate the skills you decided to load
	- Summarise the effect each one had on what you did
